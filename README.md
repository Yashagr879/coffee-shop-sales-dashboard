# ☕ Coffee Shop Sales Analytics Dashboard

An end-to-end Data Analytics project built using **MySQL** and **Power BI** to analyze coffee shop sales data and generate actionable business insights through interactive dashboards.

---

## 📌 Project Overview

This project demonstrates the complete data analytics workflow, from importing and analyzing sales data in MySQL to creating an interactive Power BI dashboard. The dashboard helps identify sales trends, store performance, product performance, and customer purchasing patterns.

---

## 🚀 Tech Stack

- MySQL
- SQL
- Power BI
- DAX
- Power Query

## 📊 Dashboard Features

- Total Sales KPI
- Total Orders KPI
- Average Order Value
- Monthly Sales Trend
- Store-wise Sales Analysis
- Product Category Analysis
- Hourly Sales Trend
- Top Selling Products
- Interactive Filters and Slicers

---

## 📈 Business Insights

- Identified monthly sales trends and revenue growth.
- Compared sales performance across multiple store locations.
- Analyzed product category contribution to overall revenue.
- Identified peak business hours based on customer transactions.
- Determined top-selling products to support inventory planning.
